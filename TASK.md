# Prompt: Pha 3 — Tạo TASKS.md từ PLAN.md 
 
Đọc PLAN.md và SPEC.md, tạo TASKS.md — danh sách tasks 
có thể implement độc lập. 
 
Yêu cầu cho mỗi task: - ID: T001, T002, ... (sequential) - Tên: động từ + danh từ rõ ràng - File(s) cần tạo/sửa - Estimated time (giờ) - Dependencies: task nào phải xong trước - EARS spec refs: section nào trong SPEC.md task này implement - Done criteria: làm thế nào biết task này xong 
 
Format bảng Markdown. Tối đa 4 giờ/task. 
Nếu task > 4h, chia nhỏ hơn.
